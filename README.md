# IT Trainer Internship Program: Becoming FullStack Developer

## Role and Goal
As your IT Trainer, my role is to guide beginners with basic Front-End knowledge to become FullStack developers through hands-on practice by completing a project.

The goal is to complete a project in one sprint (2 weeks) using an AI agent for writing and modifying code, and peer-to-peer learning with the team to understand basic CRUD operations and working with database tables.

## Format
The team selects a new captain every week on Monday. The captain organizes daily calls, agrees with the team on who takes which task, and tracks deadlines.

All decisions are made in text in the channel. If you can't solve something yourself with AI help, the captain calls the mentor for assistance.

The captain also reviews code and merges pull requests.

## Project Outcome
The result of the sprint should be launching the application via a single docker compose file from https://github.com/craftled/bordfu, using https://github.com/apitable/apitable instead of Airtable.

This is an extremely minimal and simple project aimed at removing fear of working with databases and providing just-in-time knowledge through channel discussions with participants.

Keep the project extremely minimal and simple with realistic milestones for the sprint (10 days - 2 weeks, Monday to Friday).

## First Task for Each Participant
1. Go to https://github.com/Help4Work/internship-nov-2025
2. Click the "Fork" button (top right) to create your copy.
3. In your forked repo, click on README.md, then the pencil icon to edit.
4. Add your line to the participants table below in this format: Name Surname LinkedIn profile
5. Scroll down, add a commit message like "Add myself to participants", and click "Commit changes".
6. Go back to the main repo page, click "Contribute" > "Open pull request".
7. Add a title like "Add [Your Name] to participants" and click "Create pull request".
